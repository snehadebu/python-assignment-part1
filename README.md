# Python Assignment — Part 1 (Student Grade Tracker)

## Overview

This assignment is based on basic Python concepts like loops, conditionals, strings, and lists.
I created a simple student grade tracker that cleans data, calculates results, and performs some string operations.

---

## Tasks Covered

### Task 1 — Data Cleaning

* Cleaned student names (removed extra spaces and fixed casing)
* Converted roll numbers from string to integer
* Converted marks from a string into a list
* Checked if names are valid (only alphabets)
* Printed student profile cards

---

### Task 2 — Marks Analysis

* Printed subject-wise marks with grades
* Calculated total and average marks
* Found highest and lowest scoring subjects
* Added new subjects using a while loop
* Handled invalid inputs (like wrong marks)

---

### Task 3 — Class Performance Summary

* Calculated average marks for each student
* Assigned pass/fail based on average
* Displayed a formatted table
* Counted number of pass and fail students
* Found class topper
* Calculated overall class average

---

### Task 4 — String Manipulation

* Cleaned the essay using strip()
* Converted text into title case
* Counted occurrences of the word "python"
* Replaced "python" with "Python 🐍"
* Split essay into sentences
* Printed sentences with numbering

---

## Concepts Used

* for loop, while loop
* if-elif-else conditions
* lists and dictionaries
* string functions (strip, title, split, replace, count)
* type conversion
* basic input handling

---

## How to Run

1. Open the file:
